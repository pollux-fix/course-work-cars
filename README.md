# course-work-cars

Это курсовая работа по Языкам программирования за 3 семестр
Тема: Моделирование автомобильного движения, оптимизация курсовой работы за 2 семестр

В папке vscode:
    launch.json (для конфигураций)
    settings.json
    tasks.json (для аргументов и библиотек)
( в файлах помечены места, где требуется изменить путь до какого-то файла
 это необходимо для корректной работы)

файл libfreeglut.dll - отвечает за корректность запуска

В папке include - заголовочные файлы
В папке lib - библиотеки

Библиотеки и компилятор были установлены через mingw console
