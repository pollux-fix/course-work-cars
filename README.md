# course-work-cars

Моделирование автомобильного движения, оптимизация курсовой работы за 2 семестр
==========

Описание работы
-----

Необходимо смоделировать движение автомобилей для оценки загруженности шоссе. Цель: разработка программы для моделирования автомобильного движения на языке Си с использованием фреймворка OpenGL для графической составляющей.
Необходимо проанализировать исходную программу и реализовать алгоритмическую оптимизацию. Также необходимо реализовать не менее двух машинно-независимых оптимизаций программы.
На каждом этапе необходимо выполнять профилирование для отслеживания результата.



Рабочие файлы
-----

⚪ В папке vscode:
    <li>launch.json (для конфигураций)
    <li>settings.json
    <li>tasks.json (для аргументов и библиотек)</li>

⚪ файл libfreeglut.dll - отвечает за корректность запуска

⚪ В папке include - заголовочные файлы

⚪ В папке lib - библиотеки

_Библиотеки и компилятор были установлены через mingw console_
